## About The Project

This project is a simple luck game which shows random images and stop randomly on one of them which represent their result (luck).



### Built With
* [Bootstrap](https://getbootstrap.com)
* [React](https://reactjs.org/)
* [ASP.NET Core 5](https://dotnet.microsoft.com/)



## Getting Started

### Prerequisites

* npm
  ```sh
  npm install npm@latest -g
  ```

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/AhmedAlzubaidi/ImageGame.git
   ```
2. Install NPM packages
   ```sh
   # navigate to frontend then execute
   npm install
   ```



## Usage

After running the backend and frontend app you can go ahead and create your own luck game by navigating to

```
<yourdomain>/games/create
```

then name your game and add the images you want to use you can try your game by navigating to

```
<yourdomain>/games/<id>/play
```
